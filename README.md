## ключевые особенности:

1 особенность: проект умеет рассчитывать периметр круга с помощью формулы 2 * pi * r где pi - число пи, а r - вводимый параметр.

2 особенность: проект умеет расчитывать периметр квадрата с помощью формулы n * 4 где n - вводимый параметр

3 особенность: проект способен расчитывать площадь треугольника

4 особенность: проект способен ловить исключения что не вызовет вылета при вводе текста

5 особенность: проект умеет выполнять операции сложения, вычитания, умножения, деления, и возвода в степень

6 особенность: проект умеет расчитывать площадь и периметр прямоугольника

7 особенность: доступна функция рандома

8 особенность: **возможность сканировать порты localhost**

9 особенность: **возможность узнавать ip-адреса других сайтов**

## как запустить готовый файл main.exe:
- скачайте готовый файл из релизов (самый новый: 0.2.0)
- запустите скачанный файл двойным кликом мыши
- можете оставить идеи для новых релизов
## как запустить main.py:
**что нужно: python 3 (желательно python 3.1x), c предустановленной библиотекой rich, math, random и библиотекой socket**
- скачиваем исходный код программы, кликаем на главной на надпись Code
- затем нажимаем donwnload zip
- или если у вас имеется git - откройте терминал, и введите: **git clone https://github.com/Kirill1943/Area-of-a-circle-calculation.git**
- если вы скачали через git clone - у вас появятся файлы main.py, ports.py и другие. находим файл main.py, и запускаем
- если вы скачали через интерфейс github - то у вас в загрузках появится zip архив. распакуйте его. и найдите файл main.py, после - кликаем левой кнопкой мыши 2 раза
